# Apprenticeship Vacancy API Client and Scraper

A full-blown, highly specialized API Playground, Paginated Importer, and Virtual Relational Database Studio built for UK Apprenticeship Service APIs.

## Capabilities
* **Interactive Playground**: Execute REST calls, inject custom environment variables, view syntax highlighted outputs.
* **Paginated Importer Jobs**: Set up multi-page extraction loops, configure delays, deduplicate results, trace HTTP page performance metrics.
* **Deep Reference Lookups (Detail Scraper)**: Extract sub-resource details automatically.
* **Interactive JSON Tree Explorer**: Discover JSON properties with collapsible node trees, search, and path copying.
* **Virtual Relational Tables**: Sync job data to virtual structures, search table items with SQL queries, and export structures to SQLite, MySQL, or Postgres.
