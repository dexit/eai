# Gemini Developer Mode & Custom Rules

This file documents the SuperSubAgent custom rules, developer mode constraints, and project configurations running in this session.

## System Instructions & Context
We are working on a React + Vite + TypeScript application designed to act as a Rest Client and Paginated Data Importer for the UK National Apprenticeship Service API.

## Core Mandates
1. Always maintain zero build and linting warnings or errors.
2. Ensure highly visual and tactile UI with high-contrast color schemes, custom-designed components, and full tooltips.
3. Keep logic highly modular; do not write massive unparseable single-file segments.
4. Support standard SQLite, postgresql, and MySQL exports.
